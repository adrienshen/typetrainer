Type Trainer

Instructions:
Practice touch typing. Type the short phrases that pops up before it gets too big. Game ends if you miss too many!

You can change different key sets and game modes

  • Press 1 for only homerow keys
  • Press 2 for homerow and top row
  • Press 3 for all three rows
  • Press 9 for sniper mode!
