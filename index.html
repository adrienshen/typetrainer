Type Trainer

Instructions:...
Practice touch typing. Type the short phrases that pops up and press the space bar. Type it before it gets too big or it will be a miss. Game ends on 5 misses.

You can change the mode at anytime:

  • Press 1 for only homerow keys
  • Press 2 for homerow and top row
  • Press 3 for all three rows
  • Press 9 for sniper mode!

Try it now.
